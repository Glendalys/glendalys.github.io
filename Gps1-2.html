Marko and Glendalys

Pair Programming is when two people are working together to write code or a program.

P.P. is a learning tool that teaches us to work in teams and help us to learn from each other.

The driver writes the code while the navigator review the code and comes up with ideas. The point is to learn to comunicate and build together.
